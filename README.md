# prototype-kindness

入力した言葉に対して、優しく寄り添う言葉を返すアプリのプロトタイプ。
自分のために作る、最初の完成プロジェクト。

このアプリは、入力した言葉に対して優しく寄り添うメッセージを返すアプリ。
「落ち込んだときの居場所。少しだけ心の緊張が緩む存在。」を目指す。

## 1月のゴール
画面は1つ
- 入力欄
- 送信ボタン
- 優しい返答を表示
- スマホで動作確認できる
- 思いついたらテーマカラー入れてみる
※返答は最低1パターンでもok
※実装はClaudeに任せ、設計+レビューを自分で行う

### 【今回使うもの】
- Kotlin
- Android Studio
- Jetpack Compose
- ローカルのみ（通信なし）
- 状態はメモリのみ（保存しない）

### 【優しい返答ロジック】
今回は、動作することと、
「ちゃんと優しく返してくれる」体験を最優先にする。
- 返答は少数のランダムパターン
- 入力内容の高度解析は行わない
- まずは “安心できる一言” を届けることに集中する
文章例 「お疲れさま。今日もよく頑張ったんだね。」


将来的には心理学モデルベースの返答や
より賢い応答に発展させたい。

### 【やらないこと】
- デザインこだわりすぎない
- 多機能化しない
- 設定画面作らない
- ネットワーク通信しない
- 凝ったAIロジックはやらない

### 【仕様】
#### 入力欄

  - プレースホルダ：「今の気持ちや、伝えたいことを自由に書いてください…」
  - 位置：中央
　- 文字数制限：とりあえず 200〜500文字くらい？

#### 送信ボタン

  - テキスト：「送信する」
  - 位置：入力欄の下
  - 非活性条件：入力欄が空欄/入力欄の文字数上限オーバー/送信ボタン押下後
  - 備考：返答表示までの間、テキストを「返答を考えてます」にして非活性にする

#### 返答表示

  - 位置：入力欄上
  - 文字数制限：長くても50文字以内？
  - 備考：カードの形で表示
