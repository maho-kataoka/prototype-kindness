# prototype-kindness

入力した言葉に対して、優しく寄り添う言葉を返すアプリのプロトタイプ。
自分のために作る、最初の完成プロジェクト。

このアプリは、入力した言葉に対して優しく寄り添うメッセージを返すアプリ。
「落ち込んだときの居場所。少しだけ心の緊張が緩む存在。」を目指す。

## 1月のゴール
画面は1つ
- 入力欄
- 送信ボタン
- 優しい返答を表示
- スマホで動作確認できる
- 思いついたらテーマカラー入れてみる
※返答は最低1パターンでもok
※実装はClaudeに任せ、設計+レビューを自分で行う

【優しい返答ロジック】
今回は、動作することと、
「ちゃんと優しく返してくれる」体験を最優先にする。
- 返答は少数のランダムパターン
- 入力内容の高度解析は行わない
- まずは “安心できる一言” を届けることに集中する
文章例 「お疲れさま。今日もよく頑張ったんだね。」


将来的には心理学モデルベースの返答や
より賢い応答に発展させたい。

【やらないこと】
- デザインこだわりすぎない
- 多機能化しない
- 設定画面作らない
- ネットワーク通信しない
- 凝ったAIロジックはやらない
